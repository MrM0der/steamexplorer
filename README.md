# steamexplorer
Позволяет запускать почти любую игру от имени Spacewar. (Allows you to run almost any game on behalf of Spacewar.)

```
Linux: (для этой системы вам не обязателен steamexplorer)
1. Заходим в браузер (в моём случае это LibreWolf) и вставляем в адресную строку "steam://run/480" (без ковычек).
2. Ждём установки Spacewar.
3. Заходим в коренную директорию Spacewar (У меня находится здесь ~/.steam/steam/steamapps/common/Spacewar)
4. Переименовываем steamexplorer.exe в SteamworksExample.exe и перемещаем в коренную директорию Spacewar с заменой предыдущего файла. (Вместо steamexplorer.exe вы можете использовать taskmgr.exe. Его можно найти в ~/.wine/drive_c/windows/system32, если у вас установлен Wine и запущен хотя бы один раз)

Windows:
1. Жмём Win + R и вставляем туда "steam://run/480" (без ковычек). (Либо также в браузер)
2. Ждём установки Spacewar.
3. Заходим в коренную директорию Spacewar (Находите папку в которой у вас установлен стим и потом переходите сюда Steam/steamapps/common/Spacewar)
4. Переименовываем steamexplorer.exe в SteamworksExample.exe и перемещаем в коренную директорию Spacewar с заменой предыдущего файла.
